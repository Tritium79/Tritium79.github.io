目录命名

层级 命名规则 示例 说明
根级目录 全小写，无分隔符 pages/, content/, assets/, scripts/, template/, data/ —
页面文件 全小写拉丁语 commentarii.html, silvae.html 后缀 .html
文章文件夹 帕斯卡 + 连字符 Cat-Record, Blog-Init, OA-Introduction-Translation, Article-Template 每词首字母大写，单词间连字符
文章内文件 固定为 index.html + 资源文件原名 index.html, Cat.jpg 核心文件必须为 index.html
脚本文件 全小写 build.py, build.sh —
静态资源 描述性小写 avatar.png, style.css —
字体 保留厂商原名 LXGWBright-Light.woff2 —
模板占位符 {{ double mustache }} {{ title }}, {{ content }}, {{ date }}, 存档 —
条目模板占位符 %%UPPER_PERCENT%% %%CATEGORY%%, %%TITLE%%, %%FOLDER%%, %%DATE%% build.py 内部使用
